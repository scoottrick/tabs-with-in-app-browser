Normal Behavior

This opens a page with the standard Ionic Tabs layout created with the CLI tabs template. Use this to observe the intended behavior of the layout.

Steps

  1. Tap the open button on this card.
  2. Play around with the tabs on the tab page.
  3. Notice the tabs are correctly shown as active an inactive.
 Buggy Behavior

This is intended to show issues with the Ionic Tabs layout after and In App Browser is opened. After opening the In App Browser in the app, the tabbed navigation continues to work correctly. However, the active/inactive state of the tabs is updated incorrectly.

Steps

  1. Tap the open button on this card.
  2. Wait for the in app browser to open and load.
  3. Close the in app browser ("Done" button in bottom left on iOS, "x" button in top right on Android).
  4. Play around with the tabs on the tab page.
  5. Notice the active/inactive state of the tabs is recorded incorrectly. Note: the correct components should still open in the tab layout.
